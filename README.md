## 使用方式

请访问[rck](https://fengyinchao.github.io/RCK/?path=/story/button--default)获取详细使用指南

## 开发

```
npm run storybook
```

## 发 npm 包

```
npm run build // 构建包含 es , commonjs , umd 三种格式的包
npm run publish 
```

## 发布组件使用指南至 gh-pages

```
npm run deploy-storybook
```

## 注意
- 组件 scss 样式内置于组件之中，故使用时需自行配置 scss loader
